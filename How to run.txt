1. Install old python version and create a virtual environment: 
py -3.11 -m venv venv

2. Activate the new environment: 
.\venv\Scripts\activate

3. Run the app: 
streamlit run app.py

4. To open jupyter notebook: 
python -m notebook

5. Deployed: 
https://tamil-nadu-crop-recommendation-system.streamlit.app/
